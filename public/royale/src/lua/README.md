# royale/src/lua — Lua assets

Vendor the following two files from the **WhateverRoyale addon repo** into this directory:

- `LibStub.lua` — standard WoW addon library stub (pure Lua, no WoW API)
- `LibSerialize.lua` — binary Lua serialization library (pure Lua, no WoW API)

`compat.lua` and `LibJSON.lua` are already provided in this repo.

## How to get them

From the WhateverRoyale addon's `libs/` directory (or their public GitHub repos):
- LibStub: https://github.com/Stanzilla/LibStub
- LibSerialize: https://github.com/nevcairiel/LibSerialize

Copy the single main Lua file from each into this directory.
